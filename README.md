# Real Time Analysis for Stock Market

A portfolio data consultant project where I engineer a data pipeline end-to-end for MarketPulse, a small US-based stock market analytics company.

My tech stack will include:
- Python: connects with RapidAPI for minimal latency real-time data on the stock market
- Apache Kafka: streaming the real-time stock market data
- Apache Spark: enabling analytics on the large datasets to be used
- PostgreSQL: storing the datasets for analysis 
- PowerBI: performing data analytics and making insight actionable dashboards for the company
